Cursor Auto Hide

Keep the pointer out of the way while you type.

A lightweight Chrome and Edge extension that makes text editing feel cleaner by hiding the mouse pointer while you are actively typing in editable fields.

> typing...

The pointer disappears while you type, then returns as soon as you move the mouse again.

Simple behavior

Works with standard text fields and rich-text editors without adding site-specific complexity.

One global toggle

Enable or disable it for all sites from the extension popup. No account, no sync service, no clutter.

Privacy-first

No tracking, no content collection, and no network calls. The extension only stores whether it is enabled.

Install now

Until the store listing is live, install it locally.

  1. Open chrome://extensions or edge://extensions.
  2. Turn on Developer mode.
  3. Click Load unpacked.
  4. Select the cursor-auto-hide folder.
