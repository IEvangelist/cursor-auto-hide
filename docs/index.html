Chrome + Edge extension

Hide the pointer while you type.

Cursor Auto Hide hides the mouse pointer while you type in editable fields, then brings it back the moment pointer activity resumes.

Install locally

Install in under a minute.

  1. Open chrome://extensions or edge://extensions.
  2. Turn on Developer mode.
  3. Click Load unpacked.
  4. Select the cursor-auto-hide folder.

Quick tip

Search for this name

If you use the browser extension search box, search for:

Cursor Auto Hide

After loading, use the popup to turn the extension on or off for all sites.

Privacy

No browsing data collection

The extension stores one setting: whether it is enabled. It does not collect browsing activity, typed content, or page content.
