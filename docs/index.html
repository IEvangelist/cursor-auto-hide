Install

Download and load it

Until the browser stores are live, download the packaged ZIP and load it unpacked.

  1. Download cursor-auto-hide-latest.zip.
  2. Extract the ZIP somewhere easy to find.
  3. Open chrome://extensions or edge://extensions.
  4. Turn on Developer mode.
  5. Click Load unpacked.
  6. Select the extracted cursor-auto-hide-* folder.

Search for Cursor Auto Hide after install.
