Install

Install from ZIP

Stores soon. The ZIP install works today.

  1. Download and extract cursor-auto-hide-latest.zip.
  2. Open the extensions page. Browsers block direct links to internal settings pages, so copy one and paste it into the address bar:
  3. Turn on Developer mode.
  4. Click Load unpacked and choose the extracted cursor-auto-hide-* folder.

Search for Cursor Auto Hide after install.
