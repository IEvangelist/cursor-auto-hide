Native-feeling behavior

The pointer comes back as soon as you need it.

Cursor Auto Hide is intentionally simple: it hides the pointer while you type, then immediately restores normal behavior on common pointer and focus events.

Restore event Pointer movement As soon as the mouse moves again, the normal pointer returns.
Restore event Click Pointer interaction immediately ends the hidden-pointer state.
Restore event Scroll Scrolling brings the pointer back without waiting for another keystroke.
Restore event Blur Leaving the field restores the normal pointer automatically.
Restore event Tab switch Changing visibility or focus clears the hidden-pointer state.
Restore event Idle timeout A short pause while typing restores the pointer even without movement.
